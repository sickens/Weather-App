Welcome to Weather App

The Weather App is a dynamic web application developed using the powerful Django framework for the backend and the Bootstrap framework for the frontend. It leverages the OpenWeatherMap API to provide real-time weather data and forecasts for cities worldwide. The application consists of several pages, each serving a distinct purpose. The "Current" page displays the current weather conditions, including temperature, humidity, wind speed, and a descriptive weather icon. The "Forecast" page presents a five-day weather forecast, offering hourly updates on temperature, weather descriptions, and icons. The "Precipitation" page provides detailed precipitation information, including the probability of precipitation, rainfall, snowfall, cloudiness, humidity, wind speed, and wind direction. By utilizing different API endpoints, each page retrieves and displays relevant weather data, ensuring a comprehensive and user-friendly experience.
